# Isotope

A self contained client photo gallery software.

### Features

 - [] FS based album heirarchy
 - [] Access restriction to albums
 - [] Beautiful client
 - [] Self contained binary

### Usage

```shell
go run main.go start --port 8080
# or
go build
./isotope start --port 8080
```
